# InsightAgent v1.5 🚀

InsightAgent is a smart data analysis web app powered by GPT-4 and Streamlit.

## Features
- Upload & preview datasets
- Automatic feature engineering
- Executive GPT summaries
- Strategy briefs with correlation maps
- Interactive dashboard builder
- Predictive modeling and business proposals

## Setup
```bash
pip install -r requirements.txt
streamlit run app.py
